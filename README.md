##  Project Overview
An interactive data analysis of the EU's top economies, focusing on the correlation between **GDP Growth**, **Inflation (HICP)**, **Unemployment**, and **Housing Markets**.

##  Key Insights & Features
- **Correlation Analysis:** A dynamic scatter plot to identify stagflation or overheating zones.
- **KPI Tracking:** Real-time benchmarking against the **ECB 2% Inflation Target**.
- **Interactive Highlighting:** Cross-source filtering to track specific country performance across multiple datasets.

##  Data Stack
- **Source:** Eurostat (Open Data)
- **Processing:** Excel (Data cleaning & YoY growth calculations)
- **Visualization:** Tableau Desktop 
- **Metrics:** GDP Annual Growth, HICP, Unemployment Rate, House Price Index (HPI).
